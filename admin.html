Export to your website

Every time you add or edit products here, click the buttons below to download the updated page files. Then drag those files into your website folder and push to GitHub — your live site updates automatically.

Collection Page

Generates collection.html with all your Mix Collection items.

Unique Pieces Page

Generates unique.html with your one-of-a-kind exclusive pieces.

How to update your live website:
1. Click a Download button above
2. Move the downloaded file into your website folder (where index.html is)
3. Open GitHub Desktop → you'll see the changed file
4. Type a summary like "Update collection" → Commit → Push
5. Your live site updates in ~1 minute
